The Extensions window displays the file extensions that IISxpress will not compress. It is recommended that any pre-compressed files (like GIF or PNG images), files which will not compress well (like MP3 files) or files where the client browser does not properly support compressed responses (like the case of Internet Explorer and Adobe Acrobat PDF files) should be added to the list.

To add an extension to the list click the Add button. To remove extensions from the list select the unwanted items and press the Delete button.
