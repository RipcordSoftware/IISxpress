In Search mode the wizard will search your web site directories for candidate content types. Before progressing you must select which web sites to search. The default is to search all web sites configured in IIS. If you have a lot of sites with complex directory structures then this may take a while, so the wizard allows you to elect which web sites to search.

Click Select from the list to enable the list of the available web sites, then click the checkbox next to each of the web sites you wish to search.

Click the Next button when you are happy with your selection.

Once the wizard has collected all the file information from your web sites it will send an HTTP request to IIS for each of the files in order to get a list of available content types.

If you click Back the search will stop and you will be returned to the previous page.

Once the wizard has completed searching your web site directories for candidate extensions it will display all the unique content types found. Place a check mark next to each content type you wish to exclude. You can select as many items as you wish here.

Once you are happy with your selection press the Next button to progress to the next wizard page. You can also click the Back button to return to previous pages.

The confirm page allows you to review the selections you have made on previous pages. When you click the Next button your selections will be committed to IISxpress's configuration database.
