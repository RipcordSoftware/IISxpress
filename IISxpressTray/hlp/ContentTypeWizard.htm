The Content Type Exclusion Wizard allows you to specify that responses with matching content types are to be excluded from the compression process.

The Content Type Exclusion Wizard has two modes of operation: you can search for content types or enter them manually. If you elect to search for content types the wizard will perform the following steps:
  • All your web site directories will be searched and a list of files found will be created.
  • Each file found will be requested from IIS to acquire its content type. For example, if the wizard finds a file called readme.txt it will ask IIS for this file. The response from IIS will contain the content type text/plain, this type will be added to selection list.

To specify which content types to exclude you should select to 'enter them manually now'. You are not limited to content types which already exist on your web server.
