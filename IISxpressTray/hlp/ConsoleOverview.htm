Overview

The IISxpress Console application is a command line tool which can be found in the IISxpress installation directory on our hard disk. This tool allows access to much the same functionality as the Status Window tool: you can add rules, monitor activity, enable or disable the ISAPI filter and change the compression mode. The following section outlines the command line parameters.

Command Line Arguments

add [extn | ct | uri | ip | file] Add an extension, content type, URI, IP or file exclusion rule.
remove [extn | ct | uri | ip | file] Remove an extension, content type, URI, IP or file exclusion rule.
list [extn | ct | uri | ip | file] List extension, content type, URI, IP or file rules.
filter [enable | disable] Enable or disable the ISAPI filter.
compmode [default | speed | size] Query or set the compression mode.
monitor Monitor IISxpress compression activity.

For more information on the above command line arguments consult the help provided by the console tool. For example: IISxpress help add will give detailed help on the add function.
