When in manual mode the wizard will allow you to enter any number of file extensions. The drop down box contains a list of extensions loaded from your computer's registry. You can select from this list or type extensions directly into the text area.

Once you have selected or typed an extension click the Add button, this places the extension into the list of extensions to add. You can add as many extensions as you wish here.

To remove candidate extensions simply select the extensions you wish to remove from the list and click the Remove button. Alternately to clear the list contents click the Clear button.

Once you are happy with the extensions you have entered click the Next button to progress to the next page.

The confirm page allows you to review the selections you have made on previous pages. You can use the Back button to return to previous pages to modify your selections.

When you click the Next button your selections will be committed to IISxpress's configuration database.
