IISxpress is available in two editions, please make sure you read and understand the terms under which you are licensed to use IISxpress.

IISxpress Community Edition is a feature complete product - there are no missing or reduced features. IISxpress does not contain spyware and does not inject advertising into the responses served to your clients - you just get 100% compression.

Community Edition for Windows Workstations

IISxpress is free for use on Workstation versions of Microsoft Windows. Currently this list includes Windows Vista and Windows XP Professional. For more information about the Community license please see the CommunityLicense.rtf file in the IISxpress installation directory on your computer.

Community Edition for Windows Servers

IISxpress is free for non-commercial and non-governmental use on Windows Servers (Windows Server 2003 and Windows Server 2008). If you are a private individual or non-profit organization then you can use IISxpress without requiring a commercial license. For more information about the Community license please see the CommunityLicense.rtf file in the IISxpress installation directory on your computer.

If you have any doubts about your eligibility to use the Community Edition then please email our sales team at sales@ripcordsoftware.com.

Commercial Edition for Windows Servers

To use IISxpress on Windows Servers in a commercial or governmental organization you are required to purchase a license. You are not in violation of the license if evaluating IISxpress, however when a decision is made to use the product then a license must be purchased.

For more information about the Commercial or Evaluation licenses please see either the License.rtf or EvaluationLicense.rtf files in the IISxpress installation directory on your computer.
