The system tab displays information about your computer like the version of Windows you are running, the amount of memory installed, the version of IIS installed, the CPU usage levels of IIS and IISxpress and the memory usage levels of IIS and IISxpress.

Platform

The Platform area shows the version of Windows you are using, the amount of physical memory installed and the type of CPU installed. This information is useful if you need to raise a support issue regarding IISxpress.

IIS

The IIS area shows the version of IIS installed on your computer, the version of ASP.NET installed, the amount of CPU time used by IIS and the amount of memory used.

IISxpress

The IISxpress area shows the amount of CPU time used by IISxpress and the amount of memory used. The graph window shows the recent CPU usage history of IISxpress.
