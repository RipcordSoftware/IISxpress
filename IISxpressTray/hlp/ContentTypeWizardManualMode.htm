When in manual mode the wizard will allow you to enter any number of content types. The drop down box contains a list of content types loaded from your computer's registry. You can select from this list or type content types directly into the text area.

Once you have selected or typed a content type click the Add button, this places the content type string into the list of content types to add. You can add as many content types as you wish here.

To remove candidate content types simply select the items you wish to remove from the list and click the Remove button. Alternately to clear the list contents click the Clear button.

Once you are happy with the content types you have entered click the Next button to progress to the next page.

IISxpress supports wildcard matching of content type strings in additon to exact matching. For example, to exclude all image types from compression you would enter a content type exclusion string of: image/*. To find out what the content type of a specific file is simply launch your favourite browser and request the file you are interested in. You can then view the content type returned by selecting the compressed response in the history tab of the IISxpress Status Window.

The confirm page allows you to review the selections you have made on previous pages. You can use the Back button to return to previous pages to modify your selections.

When you click the Next button your selections will be committed to IISxpress's configuration database.
