The general tab gives an overview of the status of IISxpress.

Status

The Status area shows the state of IIS and the IISxpress components. When IISxpress is running correctly IIS, the IISxpress Filter and the IISxpress Server will be shown as Running. On IIS 6 and above the filter will show as Installed until the first request for compression is processed.

You can enable/disable the IISxpress Filter and start/stop the IISxpress Server from the Configuration page.

Activity

The Activity area shows the real-time overall compression performance of IISxpress. You can view the total number of responses that have been compressed, the total size of the data before compression, the total size of the data after compression and the compression ratio.

For example, if the compression ratio is 75% then the data has been reduced by 75% to occupy only 25% of its original size.
