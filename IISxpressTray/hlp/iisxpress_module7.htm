IISxpress installs a native module for IIS 7 on Windows Vista and Windows Server 2008. The module interacts with IIS's response processing and forwards suitable responses to the IISxpress compression server component for processing.

The IISxpressNativeModule appears in the module list in the IIS Manager window on Windows Vista.
