Once you have created your ideal IISxpress configuration it is a good idea to create a backup file and store it in a safe place.

Your IISxpress configuration is stored in a file called IISxpress.config. This file is stored in your IISxpress installation directory. If you accepted the default settings when you installed IISxpress then this will be Program Files\Ripcord Software\IISxpress.

Follow these steps to backup your configuration file:

  • Stop the IISxpress service with either the Status Window or your computer's Service Manager.
  • Copy the IISxpress.config file to a safe location.
  • Restart the IISxpress service.

To restore a backup configuration file follow these steps:

  • Stop the IISxpress service with the Status Window or your computer's Service Manager.
  • Copy your backup IISxpress.config over the file in your IISxpress directory.
  • Restart the IISxpress service.
