When you use the built in IIS compression you have no indication of compression activity, effectiveness or web server loading. IISxpress gives you the tools to track all these metrics so that you can get the best performance from your compression setup.

Throughput and Compression Ratio

The General Tab of the IISxpress Status Window shows your web server compression throughtput. Here you can see the total number of responses processed, the total number of raw bytes processed, the total number of compressed bytes sent and the overall compression ratio.

Activity

All compressed responses can be viewed on the History tab of the IISxpress Status Window. Each response is displayed with its original size, compressed size and compression ratio.

Server Loading

The system status is available in the System tab of the IISxpress Status Window. Here you can monitor IISxpress's CPU load and memory usage.
