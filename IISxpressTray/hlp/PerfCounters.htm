IISxpress maintains a set of easily accessible performance counters, an example of which is shown below:
On Windows 2008, Vista and Windows 2003 performance information is captured for each configured application pool. Additionally, on Windows 2003, overall system activity is captured in the _Total column.
To display the IISxpress performance counters open the Windows PerfMon application (from Run in the Start Menu type 'perfmon' and press return), click the + button in the toolbar (or press ctrl+i) and then select the IISxpress ISAPI item (IISxpress IIS Module on Windows 2008 and Vista) in the 'Performance Item' dropdown list as shown in the screenshot below:
