Sometimes it is best to embark on a configuration process from a 'clean' start point. To do this you need to remove your existing configuration file. When IISxpress starts if it cannot find a configuration file then it creates a new one - this file will have no exclusion rules and no compression history database.

Your IISxpress configuration is stored in a file called IISxpress.config. This file is stored in your IISxpress installation directory. If you accepted the default settings when you installed IISxpress then this will be Program Files\Ripcord Software\IISxpress.

Follow these steps to 'clean' your configuration:

  • Stop the IISxpress service with either the Status Window or your computer's Service Manager.
  • Rename your existing IISxpress.config file if you wish to keep it, otherwise delete it.
  • Restart the IISxpress service.

You can restore your original configuration file by deleting the new file and renaming your original configuration file or alternately by restoring it from a previous backup.
