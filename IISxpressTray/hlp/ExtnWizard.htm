The Extensions Exclusion Wizard allows you to specify extensions to be excluded from the compression process.

The Extension Exclusion Wizard has two modes of operation: you can search for extensions or enter them manually. If you elect to search for extensions the wizard will look for candidate extensions in your web site directories. Only extensions which exist in your web site directories will be available for selection in this mode.

To specify which extensions to exclude you should select to 'enter them manually now'. You are not limited to extensions which already exist, you can specify any extension in this mode.
